# Pune Price Apartment Predictor

## A machine learning project that predicts apartment prices in Pune based on:
- Area, BHK, Parking, Balcony, Gym, Security, Swimming Pool

### Built using:
- Python
- Scikit-Learn
- Streamlit

### Model used:
- Linear Regression
- Random Forest

### Run Locally:
1.  pip install -r requirements.txt
2.  streamlit run app.py
